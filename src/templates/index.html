Sepsis Prediction API

An API for predicting sepsis in patients using machine learning models.

Introduction

Welcome to the Sepsis Prediction API. This API allows you to predict whether a patient is likely to develop sepsis based on various clinical parameters.

How to Use

To make a prediction, you can send a POST request to the endpoint `/predict` with the patient's data. The API will return a prediction indicating whether the patient is likely to develop sepsis or not.

Endpoint

POST `/predict` - Make a prediction for a single patient.
